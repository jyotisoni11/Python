# Sales Data Analysis using NumPy

## Overview
This project analyzes yearly sales data of 5 products over 12 months using NumPy.

## Features
- Monthly sales calculation
- Product-wise total sales
- Best selling product
- Worst performing month
- Growth rate calculation
- Sales increase simulation
- High revenue month filtering

## Skills Used
- NumPy
- Vectorized operations
- Boolean indexing
- Axis operations
- Modular programming

## How to Run
python sales_analysis_numpy.py
